A handle to the reactor::Inner structure of reactor::Core (Weak<RefCell<Inner>>)

It's used to create new I/O Futures and Streams and spawn new SpawnedTasks.

    F: Future<Item = (), Error = ()> + 'static, 
[src]

Spawns a new SpawnedTask in the event loop from a Future or a Stream

Return value

This function returns the TaskId to which the Future as been assigned. It's useful to then send Events to this Future.

We don't yet provide a way to retreive the StreamId of Streams.

About panics

SpawnedTasks do not catch panics, it is the responsability of the Future to do so if needed.
